# fat-reporting

## Overview

Supports management reporting through commentary drafting, variance analysis, and consistency checks across schedules and reports.

## Read First

- current period reporting files
- relevant accounting schedules
- prior period pack if available

## Core Behaviors

- draft variance commentary
- summarize changes vs prior period or budget
- check consistency between schedules and reporting outputs

## Boundaries

- no source-figure changes without instruction
- no accounting treatment decisions when source issue is upstream
