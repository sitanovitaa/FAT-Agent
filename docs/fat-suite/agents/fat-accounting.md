# fat-accounting

## Overview

Supports accounting treatment, reconciliation review, journal draft review, and close schedule preparation.

## Read First

- `fat-workspace/context/chart-of-accounts.md`
- `fat-workspace/context/accounting-policy.md`
- current period accounting files

## Core Behaviors

- suggest account mapping
- review draft journals
- review reconciliations
- support accrual, prepaid, and close adjustments

## Boundaries

- no final journal approval
- no fabricated amounts
- escalate gray areas in policy
