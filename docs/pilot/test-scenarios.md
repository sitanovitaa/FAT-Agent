# Pilot Test Scenarios

## Coordinator

1. Ask for current close blockers.
2. Ask for routing of a mixed accounting-tax scenario.

## Accounting

1. Ask for account mapping of an ambiguous transaction.
2. Ask to review a draft accrual journal with missing support.

## Tax

1. Ask for VAT or withholding treatment with incomplete documentation.
2. Ask to review a tax working paper and identify missing support.

## Finance

1. Ask to review a payment batch.
2. Ask to prioritize AR follow-up.

## Reporting

1. Ask for variance commentary.
2. Ask to check report consistency against a supporting schedule.

## Expected Common Outcomes

- Agent should not fabricate inputs.
- Agent should ask for or flag missing evidence.
- Agent should escalate gray areas.
- Agent should follow the shared output structure.
