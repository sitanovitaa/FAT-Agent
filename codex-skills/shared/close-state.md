# FAT Close State Reference

`close-status.md` uses these columns:
- `Step ID`
- `Function`
- `Task`
- `Owner`
- `Status`
- `Due Date`
- `Dependency`
- `Evidence`
- `Notes`

Allowed status values:
- `not_started`
- `in_progress`
- `blocked`
- `done`

`issue-log.md` uses these columns:
- `Issue ID`
- `Function`
- `Severity`
- `Issue`
- `Owner`
- `Blocker`
- `Next Action`
- `Due Date`
- `Status`
