# FAT Workspace

This workspace stores company context, period-based working files, reusable templates, and output artifacts for the FAT agent suite.

## Directory Rules

- `context/` stores shared company source-of-truth summaries.
- `periods/YYYY-MM/` stores period-specific work.
- `templates/` stores reusable review and draft templates.
- `outputs/` stores generated artifacts that are not tied to a single period.
